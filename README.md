\# NASA Space Dashboard



CLI applicatie gebouwd met de NASA Open API.



\## Functionaliteiten



1\. Astronomy Picture of the Day bekijken

2\. NASA afbeeldingen zoeken

3\. Asteroïden bekijken



\## Gebruikte API



NASA Open API  

https://api.nasa.gov



\## Authenticatie



De applicatie gebruikt een NASA API key via een `.env` bestand.



\## Installatie



```bash

pip install -r requirements.txt

```



Maak daarna een `.env` bestand:



```txt

NASA\_API\_KEY=je\_api\_key

```



\## Starten



```bash

python main.py

```
